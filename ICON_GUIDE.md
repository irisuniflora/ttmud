# 게임 아이콘 가이드

이 문서는 게임 내에서 사용되는 모든 아이콘을 정리한 가이드입니다.
아이콘이 겹치지 않도록 새로운 기능 추가 시 참고하세요.

---

## 스탯 아이콘

| 아이콘 | 스탯 | 용도 |
|--------|------|------|
| ⚔️ | 공격력 | 기본 공격력, 장비 공격력, 무기, 전투 관련 전반 |
| 💥 | 치명타 확률 | 크리티컬 확률, 치명타 로그 |
| 🎯 | 명중 | 치명타 데미지, 명중률, 명중 특화 |
| 👑 | 보스/전설 | 보스 데미지, 전설 토큰 |
| 💰 | 골드 | 골드 획득량 |
| 🍀 | 드랍율 | 아이템 드랍 확률 |
| 📈 | 경험치 | 경험치 보너스 |
| ⏭️ | 스킵 확률 | 스테이지 스킵 |
| 🔄 | 귀환 | 귀환 횟수 |
| 🔥 | 최종 데미지 | 문양 최종 데미지% |

## 문양 특수능력 아이콘

| 아이콘 | 능력명 | 설명 |
|--------|--------|------|
| 💢 | 파괴 분노 | 장비 파괴 시 데미지 +50% |
| 💫 | 백발백중 | 회피 무시 (모든 공격 명중) |
| 🎯 | 명중 특화 | 명중 +1500 |
| 🛡️ | 보호막 분쇄 | 보호막에 +100% 피해, 갑옷/보호막 |
| ⛏️ | 무적 파괴 | 보스 무적 상태 즉시 해제 |
| 🗡️ | 방어막 관통 | 데미지 30%가 방어막 무시 |
| ❤️ | 생명력 흡수 | 12초마다 HP 1 회복 |
| 🚫 | 치유 감소 | 보스 치유 효과 -30% |
| 💠 | 순수 데미지 | 최종 데미지 +12% (치명타 무효 보스 특화) |
| 🔱 | 추가 타격 | 공격 시 타수 +1 |

## 허용되는 중복 (맥락상 동일 의미)

다음 아이콘들은 같은 카테고리 내에서 사용되므로 중복 허용:

| 아이콘 | 용도들 | 이유 |
|--------|--------|------|
| ⚔️ | 공격력, 무기, 전투 | 모두 전투/공격 관련 |
| 💥 | 치명타 확률, 치명타 로그 | 모두 치명타 관련 |
| 🎯 | 치뎀, 명중 | 모두 정확도/타격 관련 |
| 🛡️ | 갑옷, 보호막 분쇄 | 모두 방어/보호막 관련 |
| 👑 | 보스 데미지, 전설 토큰 | 모두 최상위/보스 관련 |

## 주의사항

1. **다른 카테고리에서 겹치지 않게**: 같은 아이콘을 완전히 다른 의미로 사용 금지
2. **일관성 유지**: 한 번 정한 아이콘은 전체 게임에서 유사한 맥락으로 사용
3. **새 아이콘 추가 시**: 이 문서에 먼저 등록 후 사용

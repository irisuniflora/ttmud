# Abyss Walker - 방치형 텍스트 RPG

어둠의 심연을 걷는 자의 이야기. 웹 기반 방치형 텍스트 RPG 게임입니다.

## 🎮 게임 특징

### 핵심 시스템
- **자동 전투**: 틱마다 자동으로 데미지가 들어가는 방치형 시스템
- **스테이지 진행**: 일반 몬스터와 보스 스테이지 구조
- **환생 시스템**: 스테이지 50+ 도달 시 환생으로 영구 강화
- **영웅 고용**: 9명의 영웅을 고용하고 레벨업
- **장비 시스템**: 6개 슬롯의 장비 + 5단계 레어리티
- **스킬 트리**: 전투/경제/환생 3가지 트리
- **도감**: 몬스터/영웅/아이템 수집

### 성장 요소
- 플레이어 레벨 & 스탯
- 영웅 고용 및 레벨업
- 장비 획득 및 장착
- 스킬 트리 업그레이드
- 환생 포인트로 영구 강화

## 🚀 시작하기

### 필요 사항
- Node.js 18+ 
- npm 또는 yarn

### 설치

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:3000` 접속

### 빌드

프로덕션 빌드:
```bash
npm run build
```

빌드된 파일은 `dist/` 폴더에 생성됩니다.

## 🎯 게임 플레이 가이드

### 기본 플레이
1. 게임이 자동으로 시작됩니다
2. 틱마다 자동으로 몬스터에게 데미지를 줍니다
3. 골드를 모아 영웅을 고용하고 레벨업하세요
4. 장비를 획득하고 장착하여 강해지세요

### 영웅 시스템
- 영웅은 골드로 고용 가능
- 각 영웅은 고유한 DPS를 가집니다
- 레벨업하면 더 강해집니다
- 특정 레벨마다 스킬이 해금됩니다

### 장비 시스템
- 몬스터 처치 시 확률적으로 드랍
- 6개 슬롯: 무기, 갑옷, 장갑, 신발, 목걸이, 반지
- 5단계 레어리티: 일반 < 레어 < 에픽 < 전설 < 신화
- 레어리티가 높을수록 더 많은 스탯 보유

### 스킬 트리
- **전투 트리**: 크리티컬, 공격력, 영웅 강화
- **경제 트리**: 골드, 드랍률, 경험치 증가
- **환생 트리**: 영구 강화 (PP 사용)

### 환생 시스템
- 스테이지 50+ 도달 시 가능
- PP(Prestige Points) 획득
- PP로 영구 업그레이드 구매
- 게임을 처음부터 시작하지만 더 강한 상태로

# 패치노트 v1.1.0

## 🎮 신규 콘텐츠

### 봉인구역 시스템
- **레이드 보스**: 7단계 난이도 (일반 → 다크)
- **문양 시스템**: 4개 슬롯, 등급별 능력치 강화
- **펫 시스템**: 자동 전투 보조 (골드/경험치/데미지 버프)
- **보스코인 상점**: 특별 아이템 구매 가능

### 월드보스 시스템
- **출현 시간**: 매일 00:00 ~ 12:00 (한국시간)
- **전투 방식**: 60초 데미지 딜링
- **보상**: 순위별 보스코인 지급
- **경매**: 월드보스 종료 후 10분간 진행

### 일일 충전 시스템
- 매일 자정(한국시간) **봉인구역 도전권 2개** 자동 지급
- 로그인 시 자동으로 수령

---

## ⚖️ 게임 밸런스

### 봉인구역 도전권
- **변경 전**: 10층 보스 처치 시 50% 확률로 1~3개 드랍
- **변경 후**: 모든 보스 처치 시 10% 확률로 1개 드랍
- **추가**: 매일 자정 2개 자동 충전

### 장비 시스템
- 장비 드랍 시 **5% 확률로 최대 옵션** 생성
- 장비 강화 시 **크리티컬 확률/데미지는 강화 효과 제외**
- 장비 옵션에 **최소/최대값 백분율 표시** 추가

### 몬스터 도감 보너스
- 희귀/전설 몬스터 수집 보너스 **정상 작동 확인**
- 각 100층 구간별 독립 계산

## 🎨 UI/UX 개선

### 스탯 표시
- **방생 보너스** 툴팁 추가 (구간 표시: 예시 "0~4층 구간에 적용")
- **몬스터 감소** 장비+도감 보너스 통합 표시
- 체력퍼뎀, 도트 데미지, 스킵 확률 등 히어로 버프 가시성 개선

### 장비 정보
- 등급별 스탯 범위 상세 표시
- 티어 시스템 설명 추가 (50층마다 1.2배)
- 장비 강화 시스템 안내 추가
- 옵션 백분율(%) 표시로 직관성 향상

### 스킬 트리
- 전체 스킬 한 페이지에 표시
- 레이아웃 개선 및 가독성 향상

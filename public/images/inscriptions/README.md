## 🎨 이미지 사양
- **형식**: PNG (투명 배경 권장)
- **권장 크기**: 128x128px ~ 256x256px
- **스타일**: 판타지 마법진/문양 느낌
- **색상**: 각 문양의 특성을 반영한 색상 추천
  - rage: 붉은색 (분노)
  - precision: 노란색/금색 (정밀)
  - shadow: 검은색/보라색 (그림자)
  - chaos: 보라색/핑크색 (혼돈)
  - decay: 녹색/독 색상 (부패)
  - crush: 주황색/갈색 (분쇄)
  - void: 검은색/자주색 (공허)
  - thirst: 진홍색 (갈증)
  - destruction: 빨간색/검은색 (파괴)
  - eternity: 흰색/금색 (영원)

## 📌 참고사항
- 100층마다 패턴이 반복됩니다 (101~110층은 다시 분노, 111~120층은 정밀 등)
- 보스 처치 시에만 드랍됩니다
- 100층 구간마다 드랍율이 2배씩 증가합니다

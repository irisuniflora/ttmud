# 아이템 이미지 폴더

이 폴더에 아이템 이미지 파일을 넣어주세요.

## 아이템 슬롯별 파일:

### 기본 슬롯 이미지
- `weapon.png` - 무기
- `armor.png` - 갑옷
- `gloves.png` - 장갑
- `boots.png` - 신발
- `necklace.png` - 목걸이
- `ring.png` - 반지

### 등급별 이미지 (선택사항)
등급별로 다른 이미지를 사용하려면:
- `weapon_common.png` - 일반 무기
- `weapon_rare.png` - 레어 무기
- `weapon_epic.png` - 에픽 무기
- `weapon_legendary.png` - 전설 무기
- `weapon_mythic.png` - 신화 무기

## 권장 사양:
- 형식: PNG (투명 배경 권장)
- 크기: 128x128px 또는 256x256px
- 용량: 각 파일 200KB 이하 권장

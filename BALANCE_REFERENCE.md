## 1. 경험치 시스템
**파일**: `src/data/gameBalance.js`

### 레벨업 필요 경험치
```
필요경험치 = 100 × (1.1 ^ (레벨 - 1))
```

| 레벨 | 필요 경험치 |
|------|------------|
| 1 | 100 |
| 10 | 236 |
| 20 | 612 |
| 50 | 10,672 |
| 100 | 1,252,783 |

### 경험치 획득량
- **기본 경험치**: 몬스터당 10
- **레벨업 시 공격력 증가**: +5

### 경험치 보너스 소스
- 스킬 (빠른 성장): 레벨당 +10%
- 영웅 (현자): 기본 15% + 등급당 10%
- 장비 (expBonus): 등급별 상이

## 2. 골드 시스템
**파일**: `src/data/gameBalance.js`, `src/data/monsters.js`

### 골드 획득 공식
```
기본골드 = 몬스터 maxHP (체력 = 골드)
최종골드 = 기본골드 × (1 + 총골드보너스% / 100)
```

### 몬스터 골드 (층 기반)
```
일반 몬스터: HP 기준 (= 100 × 1.05^(층-1))
보스: HP × 20 기준
```

| 층 | 일반 HP/골드 | 보스 HP/골드 |
|----|-------------|-------------|
| 1 | 100 | 2,000 |
| 10 | 155 | 3,100 |
| 50 | 1,147 | 22,940 |
| 100 | 13,150 | 263,000 |

### 골드 보너스 소스
| 소스 | 효과 |
|------|------|
| 스킬 (골드 마스터) | 레벨당 +10% |
| 영웅 (골드 머천트) | 기본 10% + 등급당 8% |
| 장비 (goldBonus) | 등급별 상이 |
| 유물 (황금의 예언서) | 레벨당 +70% (곱적용) |
| 유물 (기적의 성배) | 레벨당 10% 확률로 골드 10배 |

## 3. 몬스터 시스템
**파일**: `src/data/monsters.js`, `src/data/gameBalance.js`

### 몬스터 HP 공식
```
일반 몬스터 HP = 100 × (1.05 ^ (층 - 1))
보스 HP = 일반 HP × 20
```

| 층 | 일반 HP | 보스 HP |
|----|---------|---------|
| 1 | 100 | 2,000 |
| 10 | 155 | 3,100 |
| 50 | 1,147 | 22,940 |
| 100 | 13,150 | 263,000 |
| 200 | 1,729,258 | 34,585,160 |

### 특수 몬스터 HP 배율
| 유형 | HP 배율 | 출현 확률 |
|------|---------|----------|
| 희귀 몬스터 | ×3 | 5% (수집 안됨 시) |
| 전설 몬스터 | ×10 | 2% (희귀 수집 후) |
| 희귀 보스 | ×5 (= 일반보스의 5배) | 5% |
| 전설 보스 | ×20 (= 일반보스의 20배) | 2% (희귀 수집 후) |

### 층당 몬스터 수
```
기본 몬스터 수 = 40
100층 구간마다 +10마리

1-100층: 40마리
101-200층: 50마리
201-300층: 60마리
...
```

## 5. 데미지 계산
**파일**: `src/game/GameEngine.js` (calculateTotalDPS)

### 데미지 계산 순서
```
1. 기본 공격력 = player.stats.baseAtk (시작: 10)
2. + 장비 고정 공격력
3. × (1 + 스킬 공격력% / 100)
4. × (1 + 장비 공격력% / 100)
5. + 영웅 공격력 × (1 + 동료강화 스킬% / 100)
6. × (1 + 방생 보너스% / 100)
7. × (1 + 보스 도감 보너스% / 100)
8. × (1 + 유물 보유개수 × damagePerRelic% / 100)
9. × (1 + 파멸의칼날% × 데미지유물배율 / 100)
10. [보스한정] × (1 + 거인학살자% × 데미지유물배율 / 100)
11. + 체력% 데미지 (다크 리퍼)
12. + 도트 데미지 (아크메이지)
13. 크리티컬 적용
```

### 크리티컬 시스템
```
기본 크리티컬 확률: 5%
기본 크리티컬 데미지: 150%

크리티컬 확률 캡: 100%
초과분 변환:
  - 100~200% 구간: 초과 1%당 치뎀 +3%
  - 200% 이상: 초과 1%당 치뎀 +5%
```

### 특수 데미지
| 영웅 | 효과 |
|------|------|
| 다크 리퍼 | 확률(기본10% + 별당2%)로 maxHP의 (10% + 등급당5%) 추가 피해 |
| 아크메이지 | 총 데미지의 (20% + 등급당15%) 도트 데미지 |

## 9. 유물 시스템
**파일**: `src/data/prestigeRelics.js`, `src/data/formulas.js`

### 유물 가챠 비용
```
비용 = ceil(이전비용 × 1.4)
시퀀스: 1 → 2 → 3 → 5 → 7 → 10 → 14 → 20 → 28 → 40 → ...
```

### 유물 업그레이드 비용
```
기본비용 = floor(1 + 현재레벨 × 0.5)
최종비용 = max(1, 기본비용 × (1 - 비용감소% / 100))
최대 감소: 90%
```

### 유물 보너스 적용 방식
**중요: 모든 유물 확률/수치 보너스는 곱적용입니다.**
```
최종 효과 = 기본값 × (1 + 유물보너스 / 100)

예시:
- 기본 10% + 유물 90% → 10% × 1.9 = 19% (O)
- 기본 10% + 유물 90% → 100% (X, 합적용 아님)
```

### 주요 유물 효과 (레벨당, mythic 기준)
| 유물 | 효과 | 최대레벨 |
|------|------|---------|

## 밸런스 고려사항

### 주요 밸런스 포인트

1. **유물 효과는 곱적용**: 합적용이 아님! 기본값 × (1 + 보너스/100)
2. **몬스터 수 감소**: 여러 소스에서 중복 적용 → 최소 5마리 하드캡
3. **층 스케일링**: HP 1.05배로 완만한 성장
4. **문양 등급 보정**: 고층에서 고급 등급 확률 복리 증가

### 밸런스 조정 시 참고

- 모든 % 보너스는 **합연산** 후 적용 (유물 제외)
- 유물 효과 중 일부는 **곱연산** (방생 보너스, 보스 도감 등)
- 장비 강화는 크리티컬 스탯 제외
- 최소 몬스터 수: 5마리 (하드캡)
- 보스 제한시간 기본: 20초
